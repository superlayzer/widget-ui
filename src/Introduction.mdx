<Title>Widget UI</Title>
<Subtitle>Design system for building widgets that run inside Layzer's MCP App sandbox</Subtitle>

Widget UI is a design system for [Layzer](https://layzer.ai) MCP App widgets — a Layzer-maintained fork of [`@openai/apps-sdk-ui`](https://github.com/openai/apps-sdk-ui). It provides styling foundations with Tailwind, CSS variable design tokens, and a library of well-crafted, accessible components. Widget UI includes:

- **Design tokens** &ndash; for colors, typography, spacing, sizing, shadows, surfaces, and more.
- **Tailwind 4** &ndash; pre-configured with Widget UI's design tokens.
- **Component library** &ndash; high-quality components built on top of Radix for consistent accessibility patterns.
- **Utilities** &ndash; for dark mode, responsiveness layouts, and more.
